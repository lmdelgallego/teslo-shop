## TesloDB

1. Levantar la base de datos
```bash
docker compose up -d
```

2. Install dependencies
   ```bash
   yarn install
   ```

3. Clone ```.env.example``` and rename to ```.env```

4. Change environment variables.

5. Run dev ```yarn start:dev```
